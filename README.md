# Lost & Found Portal

This project is a full-stack Lost & Found portal with:
- Frontend: React
- Backend: FastAPI (Python)
- Database: MySQL
- ML Service: YOLO/MobileNet-based image feature extraction

Subfolders:
- `frontend/` – React app
- `backend/` – FastAPI backend
- `ml_service/` – Image feature extraction service
- `sql/` – Database schema
