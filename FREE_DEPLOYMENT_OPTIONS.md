## ğŸš€ HOW TO DEPLOY WITH RENDER (Simple Steps)

### Step 1: Create Render Account (1 minute)
```
1. Go to https://render.com
2. Click "Sign up"
3. Use your email or GitHub account

```

### Step 2: Connect Your GitHub Repository (2 minutes)
```
1. Login to Render
2. Click "New +"
3. Select "Web Service"
4. Connect your GitHub
5. Select your "Lost-found-portal" repository
```

### Step 3: Configure Your Service (5 minutes)
```
1. Name: "lost-found-portal"
2. Environment: "Docker"
3. Build command: "docker-compose build"
4. Start command: "docker-compose up"
5. Click "Deploy"
```

### Step 4: Add Database (3 minutes)
```
1. Click "New +"
2. Select "PostgreSQL"
3. Name it
4. Click "Create Database"
5. Copy connection string
```

### Step 5: Add Environment Variables (3 minutes)
```
1. Go to your web service
2. Click "Environment"
3. Add variables:
   - DATABASE_URL: (from PostgreSQL)
   - JWT_SECRET_KEY: (your secret)
   - VITE_API_URL: (your app URL)
   - ... (others)
4. Save and redeploy
```

### Step 6: Done! Your app is live!
```
Your URL: https://lost-found-portal.onrender.com
```
